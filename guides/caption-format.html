Which program should I use for caption making?

You can use any image editing program that allows you to add text, but ideally, you should be able to edit that text later if you feel like changing the format or writing later. I use Canva, but any other program should work just fine.

Make it readable!

Captions should be readable, in other words, when you see the picture, the text should be easy to read.
